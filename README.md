# Grades Calculator - Java Project

A Java application with JavaFX GUI for calculating and managing student grades. Features database integration with SQLite.

##  About the Project

This project demonstrates:
- Java GUI development with JavaFX
- Database operations with SQLite
- Object-oriented programming
- File I/O operations
- Input validation and error handling

##  How to Run

### Requirements:
- Java JDK 8 or higher
- JavaFX SDK (for Java 11+)

compilation:
javac -d bin -cp "lib/*" src/application/*.java


Run:

java -cp "bin;lib/*" application.GCFX
